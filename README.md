# currency-converter

A simple Vue currency converter app made using the [ExchangeRate-API](https://www.exchangerate-api.com/) and the axios npm package.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
